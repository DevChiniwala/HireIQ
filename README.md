> ### 📜 The Manifesto: Hiring Is a High-Stakes Intelligence Problem.
> The traditional hiring process is broken. It's a storm of biased intuition, manual drudgery, and missed opportunities. Recruiters drown in resumes while top candidates slip through the cracks. **HireIQ** is the lighthouse in the storm — an autonomous, AI-driven hiring agent engineered to dissect talent with surgical precision. By combining advanced language models, a structured scoring engine, and real-time RAG retrieval, HireIQ transforms recruitment from a game of chance into a **science of strategic talent acquisition**. ## ✨ Code Spotlight: The Anatomy of Explainable Scoring

This is where HireIQ transcends simple keyword matching. The system **forces the AI to justify its entire reasoning process**, providing a clear, auditable analysis for every candidate.

```python
# utils.py — Explainable Scoring Engine
class RequirementMatch(BaseModel):
    requirement: str
    match_status: bool
    evidence: str          # Direct quote or "NOT FOUND" from resume

class ExplainableCandidateScore(BaseModel):
    name: str
    overall_score: int     # 0–100
    summary: str
    requirement_analysis: List[RequirementMatch]

def score_candidate_explainable(
    job_description: str,
    resume_text: str,
    weighted_requirements: Dict,
    llm: BaseChatModel
) -> ExplainableCandidateScore:
    """
    Scores a candidate with Explainable AI — every deduction is evidence-linked.
    Uses LangChain's with_structured_output to bind LLM output to a Pydantic model.
    """
    prompt = """
    TASK: Evaluate a candidate's resume against a job description and weighted requirements.
    Output MUST be a single, valid JSON object matching the ExplainableCandidateScore schema.

    SCORING RULES:
    1. Start at 100 points.
    2. For each requirement, if no direct evidence is found, deduct points based on weight.
    3. Provide EXACT quoted evidence or "NOT FOUND — [reason]" for every requirement.
    4. overall_score = final calculated score after all deductions.

    WEIGHTED REQUIREMENTS: {weights}
    JOB DESCRIPTION: {jd}
    RESUME TEXT: {resume}
    """
    # Magic: LangChain binds LLM output directly to the Pydantic model
    structured_llm = llm.with_structured_output(ExplainableCandidateScore)
    chain = ChatPromptTemplate.from_template(prompt) | structured_llm

    return chain.invoke({
        "weights": json.dumps(weighted_requirements),
        "jd": job_description,
        "resume": resume_text[:6000]  # Truncated for context window
    })
```

## 🛠️ Ignition Sequence: Activate HireIQ

#### 1. Clone the Repository
```bash
git clone https://github.com/iamdev0707/HireIQ.git
cd HireIQ
```

#### 2. Set Up the Environment
```bash
python -m venv venv
# Windows:       venv\Scripts\activate
# macOS / Linux: source venv/bin/activate

pip install -r requirements.txt
```

#### 3. Configure API Keys
Create a `.streamlit/secrets.toml` file — this is your configuration hub:
```toml
# .streamlit/secrets.toml
GROQ_API_KEY = "gsk_..."
```

#### 4. Execute
```bash
streamlit run app.py
```

Your HireIQ dashboard will launch at `http://localhost:8501` 🚀
